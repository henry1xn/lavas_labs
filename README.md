# Lavas Labs Frontend

Lavas Labs 前端页面集合，包含登录、注册、KYB认证、仪表板和钱包等页面。

## 页面列表

- `index.html` - 登录页面
- `signup.html` - 注册页面（黑红配色）
- `kyb.html` - KYB 企业认证页面
- `dashboard.html` - 仪表板页面
- `wallet.html` - 钱包页面

## 特性

- 多语言支持（中文、英文、日文、德文、越南文）
- 响应式设计
- 现代化 UI 设计
- 使用 MiSans 字体

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript

## 使用方法

直接在浏览器中打开对应的 HTML 文件即可查看效果。
